AE制作水墨片头P1

2017-04-05

AE–水墨片P1(笔记总结)

操作环境

AfterEffects CC 2017，mac

所用快捷键

功能 快捷键 备注
导入 cmd+i inport
复制 cmd+d
时间再设置 option+cmd+t Enable Time Remapping
预合成 cmd+shift pre-compose
新建固态 cmd+y solid setting

所需素材

  • 水墨素材（tga等格式）
  • 视频/图片素材
  • 固态背景层

主要步骤

水墨素材去背景（抠图）–>素材叠加

抠图

  1. 新建合成，命名为“ink_01”;

  2. 导入（cmd+i）水墨素材；

  3. 使用钢笔工具，勾出水墨素材需要显示的部分；

  4. 选择水墨素材，鼠标单击右键，选择“色彩校正（color correction）–> 通道（channel）–> 反转（invert）”；（或者直接在右边“效果（effect）面板搜索框输入“反转（invert）”）；

  1. 新建固态层（solid setting），选择填充颜色为黑色，并将其命名为“bg_01”，放在水墨素材下层；

  2. 选择水墨素材，鼠标单击右键，选择“色彩校正（color correction）–>色相/饱和度（Hue/Saturation）”，调节主饱和度参数（Master Saturation）、主亮度参数（Master Lightness）；

  3. 选择水墨素材，鼠标单击右键，选择“色彩校正（color correction）–> 色阶（levels）”，调节输入的黑色和白色数量，使黑白对比明显；

叠加

  1. 新建合成（cmd+n），命名为“01”;

  2. 将“ink_01”拖拽进“inkok_01”;

  3. 对“ink_01”进行复制（cmd+d），命名为“ink_01_luma”并放在“ink_01”下层；

  4. 选择“ink_01_luma”，选择“轨道蒙版（Track Matte）–>亮度蒙版（Luma Matte）”；

  5. 对“ink_01”和“ink_01_luma”进行预合成（shift+cmd+c）,命名为“inkok_01”;

  6. 导入准备好的视频素材，将其置于“inkok_01”下层；

  7. 选择视频素材，选择“轨道蒙版（Track Matte）–>透明度蒙版（Alpha Matte）”；

微调

  1. 新建合成（cmd+n），命名为“final”；

  2. 将“01”拖拽到轨道面板中，对“01”进行复制（cmd+d）,并命名为“01-1”；

  3. 对“01-1”进行放大（快键键“s”），但不能太大；

  4. 选择“01”，选择“快速模糊(fast blur)”,对模糊数值进行适当地调节；
  5. 选择“01”，选择“色彩校正（color correction）–> 色阶（levels）/色彩平衡（color balance）”等参数进行调节；
  6. 最后将“01”和“01-1”预合成，命名为“01_ok”；
