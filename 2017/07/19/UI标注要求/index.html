UI标注要求

2017-07-19

以下的位置都是相对位置: 即相对于其他控件或者屏幕边缘的位置

标签字体标准

  • 所有的字体大小和颜色都要标注

输入框标准

  • 输入框都要标注出大小和边框颜色

  • 输入框内的字体大小和颜色

  • 输入框的宽度如果能够根据边距确定就不需要标注宽度,否则要标准

图片控件

  • 图片控件都要标注出大小和位置

  • 内容不唯一的图片不需要切图, 内容唯一的图片需要切除 icon

按钮的标准

  • 按钮控件都要标注出大小和位置

  • 按钮需要切除 正常 和 选中 两种状态的图片

  • 按钮的标题的字体大小和颜色

其他

  • 尽量只需要切独立的小图片, 不需要切包含其他控件的大背景框之类的图片.
  • 大背景框需要标注出大小和位置, 及其他的附属属性: 颜色, 是否圆角, 圆角半径等
